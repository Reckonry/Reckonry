# Support

Reckonry is early-stage open-source software for verifiable financial ledgers and professional review workflows.

## Where To Ask

- Use GitHub issues for reproducible bugs, feature requests, importer requests, and country-specific tax support requests.
- Use GitHub discussions if enabled for broader questions and design discussion.
- Use security reporting guidance in [SECURITY.md](SECURITY.md) for vulnerabilities.

## Before Asking

- Search existing issues and documentation.
- Remove all private financial data, account identifiers, real exchange exports, and unredacted logs.
- Include Reckonry version, OS, .NET version, command used, expected behavior, and actual behavior for bugs.

## What Reckonry Support Is Not

Reckonry support is not tax, legal, accounting, financial, or investment advice. Validate all outputs with qualified professionals before relying on them.
