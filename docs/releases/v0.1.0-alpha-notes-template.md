# Reckonry v0.1.0-alpha Release Notes Template

## Summary

Reckonry `v0.1.0-alpha` is an early CLI-first release for validating a safe,
synthetic digital asset ledger reconstruction workflow.

This release is alpha software. It is not a tax calculator, filing product,
accounting system, hosted API, or legal/tax advice.

## Included

- Canonical ledger v1 foundations.
- Binance CSV importer foundation.
- Public synthetic demo workflow.
- Ledger validation and integrity reporting.
- Binance Italy reconciliation demo module.
- Italy RW professional review outputs.
- Tax Dossier PDF generation for professional review.
- Bundled assembly discovery for installed Reckonry modules.
- Architecture boundary tests.

## Explicitly Not Included

- Tax filing.
- Legal, tax, accounting, financial, or investment advice.
- Stable external SDK or external plugin loading.
- Production API support.
- Authentication, database persistence, or hosted workflows.
- NuGet package publishing.
- Complete Binance coverage.
- Complete multi-provider or multi-country coverage.

## Artifacts

- `reckonry-cli-v0.1.0-alpha.tar.gz`
- `reckonry-cli-v0.1.0-alpha.tar.gz.sha256`

## Verification

Before publishing:

```bash
dotnet build Reckonry.sln
dotnet test Reckonry.sln
sha256sum reckonry-cli-v0.1.0-alpha.tar.gz
```

## Known Limitations

- Requires .NET SDK/runtime compatible with `net10.0`.
- CLI is the supported public alpha surface.
- `Reckonry.Api` is experimental and not part of the supported public alpha workflow.
- Placeholder importer projects are discoverable but do not parse real files.
- Demo data is synthetic and proves one workflow only.
- Professional review outputs require qualified human validation.

## Upgrade Notes

No stable upgrade guarantees exist before `v1.0.0`. Breaking changes must be
listed in `CHANGELOG.md`.
