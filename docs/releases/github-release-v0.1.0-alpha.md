# Reckonry v0.1.0-alpha

Release date: 2026-07-04

Reckonry `v0.1.0-alpha` is the first public alpha of Reckonry: CLI-first,
local-first infrastructure for reconstructing, validating, auditing,
reconciling, and reviewing digital asset ledger evidence.

This is alpha software. Reckonry is not a tax calculator, filing product,
accounting system, hosted service, or source of legal/tax certainty.

Canonical release notes:

- `docs/releases/v0.1.0-alpha.md`

## Quick Demo

```bash
dotnet build Reckonry.sln
dotnet test Reckonry.sln
scripts/demo.sh
```

Expected alpha result: `NOT READY FOR FILING`.

That is intentional. The demo uses synthetic data and shows that missing
professional inputs are visible and are not invented.

- Public demo data is synthetic.
