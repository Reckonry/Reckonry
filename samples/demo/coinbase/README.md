# Coinbase Synthetic Demo Sample

This folder contains fake Coinbase-style CSV data for the public Reckonry demo.

Every row, timestamp, asset quantity, fiat value, note, and identifier is synthetic.
The files are safe to commit publicly and must not be replaced with real exchange
exports or private financial records.

The sample intentionally includes one unsupported row so Reckonry can demonstrate
unknown row preservation through `exceptions.csv`.
