# Binance Samples

This directory contains fake Binance-style CSV exports for importer development and documentation examples.

Do not commit private account data, API keys, transaction IDs that identify real accounts, or personally identifying information.

Current fake samples:

- `deposits.csv`
- `withdrawals.csv`
- `spot-trades.csv`
- `conversions.csv`
- `rewards.csv`
- `unknown.csv`
